{% if lang == 'ko' %}이 통계는 어떻게 해석하나요?{% else %}How to Read These Stats{% endif %}
{% if lang == 'ko' %}

통계 카드를 보는 법:

  • 플레이 횟수 — 지금까지 시작한 게임의 총 횟수 (이긴 게임 + 진 게임 + 포기한 게임 포함).
  • 승률 — 이긴 게임 ÷ 플레이 횟수. 70% 이상이면 매우 우수한 편입니다.
  • 현재 연속 기록 — 데일리 챌린지를 며칠 연속 풀었는지. 한 번 놓치면 0으로 초기화됩니다.
  • 최고 연속 기록 — 이제까지 도달한 가장 긴 연속 기록.

추측 분포 차트는 게임당 평균 몇 번 만에 풀었는지를 보여줍니다. 막대가 3~4번 사이에 쏠려 있으면 효율적인 플레이어이고, 5~6번 쪽으로 분산되어 있다면 시작 단어 전략이나 글자 빈도 직관을 다듬을 여지가 있습니다.

전략 가이드는 단어 게임 트릭 10가지와 최고의 시작 단어를 참고하세요.

{% else %}

What each stat means:

  • Played — Total games you have started so far (wins, losses, and give-ups all counted).
  • Win % — Wins divided by games played. Above 70% is considered very strong.
  • Current Streak — How many days in a row you have solved the Daily Challenge. Resets to 0 if you miss a day.
  • Best Streak — The longest streak you have ever reached.

The guess-distribution chart shows the average number of guesses it took to solve each game. If your bars are concentrated at 3 or 4 guesses, you are an efficient player. If they are spread toward 5 or 6, there is room to sharpen your opening-word strategy or letter-frequency instincts.

{% if lang == 'ko' %}통계가 이 기기에만 저장되는 이유{% else %}Why Stats Stay Local to This Device{% endif %}
{% if lang == 'ko' %}

WordMaster는 의도적으로 통계를 이 브라우저에만 보관합니다. 서버로 전송하지 않습니다.

이유:

  • 계정이 필요 없습니다 — 가입, 비밀번호, 이메일 인증 없음. 그냥 플레이하면 됩니다.
  • 개인정보 보호 — 게임 기록은 본인 외 누구도 볼 수 없습니다. WordMaster 운영자도 모릅니다.
  • 트레이드오프 — 브라우저 데이터를 지우거나 다른 기기로 옮기면 통계가 이전되지 않습니다. 이건 의도된 설계이지 버그가 아닙니다.
{% else %}

WordMaster deliberately keeps your stats in this browser only. They are never sent to any server.

Why:

  • No account needed — No signup, password, or email verification. Just play.
  • Privacy — Your game history is visible only to you. Not even the operator of WordMaster can see it.
  • Tradeoff — If you clear browser data or switch devices, your stats do not carry over. This is intentional, not a bug.
