{% if lang == 'ko' %} 글자를 입력하거나 클릭해 단어를 만들고 Enter를 눌러 제출하세요. 각 추측 후 타일 색이 바뀝니다: 초록색 = 올바른 위치, 노란색 = 잘못된 위치, 회색 = 단어에 없음. 6번 안에 정답을 찾아야 합니다. 자세한 내용은 전체 가이드를 참조하세요. {% else %} Type or click letters to form a word, then press Enter to submit. After each guess, the tiles change color: Green = correct position, Yellow = wrong position, Grey = not in the word. You have 6 attempts to find the answer. See our full guide for more detail. {% endif %}

{% if lang == 'ko' %} 모든 모드에서 6번의 추측이 주어집니다. 이것은 클래식 단어 게임의 표준입니다 — 좋은 전략으로 풀 수 있을 만큼은 충분하면서도 흥미를 유지할 만큼 도전적입니다. {% else %} All modes give you 6 guesses. This is the classic word-game standard — enough to be solvable with good strategy, but challenging enough to stay exciting. {% endif %}

{% if lang == 'ko' %} 데일리 모드는 전 세계 모든 사람이 하루에 같은 한 단어를 받습니다. 자정에 초기화됩니다. 하루에 한 번만 플레이할 수 있어 다른 플레이어와 점수를 비교할 수 있습니다.

무제한 모드는 제한 없이 매 게임 후 새로운 랜덤 단어를 제공합니다. 연습하거나 한 게임으로 부족할 때 완벽합니다! {% else %} Daily mode gives everyone worldwide the same one word per day. It resets at midnight. You can only play once per day, so your score is comparable to other players.

Unlimited mode gives you a fresh random word after every game with no restrictions. Perfect for practice or when one game isn't enough! {% endif %}

{% if lang == 'ko' %} 힌트 버튼은 숨겨진 단어의 정의와 품사를 공개합니다. 게임당 힌트 한 번만 사용할 수 있습니다. 힌트를 사용해도 승리를 방해하지 않습니다 — 범위를 좁히는 데 도움이 되는 어휘 단서를 제공할 뿐입니다. {% else %} The Hint button reveals the definition and part of speech of the hidden word. You get one hint per game. Using a hint doesn't prevent you from winning — it just gives you a vocabulary clue to help narrow things down. {% endif %}

{% if lang == 'ko' %} 통계(플레이 횟수, 승률, 현재 연속 기록, 최고 연속 기록, 추측 분포)는 브라우저에 로컬로 저장됩니다. 통계 페이지를 방문해 확인하세요. 통계는 기기별로 저장됩니다 — 브라우저를 바꾸거나 브라우저 데이터를 지우면 이전되지 않습니다. {% else %} Your stats (games played, win rate, current streak, best streak, and guess distribution) are stored locally in your browser. Visit the Stats page to view them. Stats are saved per device — they won't carry over if you switch browsers or clear your browser data. {% endif %}

{% if lang == 'ko' %} 쉬움 모드는 4글자 단어를 사용합니다 — 짧고 간단해 초보자나 어린 플레이어에게 좋습니다.

어려움 모드는 6글자 단어를 사용합니다 — 더 길고 흔하지 않아, 진정한 도전을 원하는 경험 많은 어휘 애호가에게 적합합니다. {% else %} Easy mode uses 4-letter words — shorter and simpler, great for beginners or younger players.

Hard mode uses 6-letter words — longer and less common, for experienced vocabulary enthusiasts who want a real challenge. {% endif %}

{% if lang == 'ko' %} 데일리 보관함 페이지에서 지난 30일간의 데일리 단어와 각 단어의 정의를 볼 수 있습니다. 학습하거나 놓친 날 확인하기에 좋습니다! {% else %} The Daily Archive page lists the past 30 days of daily words, along with each word's definition. Great for learning, or if you missed a day! {% endif %}

  • 고유 명사 (이름, 장소, 브랜드)

  • Abbreviations or acronyms
  • Very rare, archaic, or highly technical words
  • Hyphenated words
If you believe a common word is missing, please let us know! {% endif %}

{% if lang == 'ko' %} 아니요. 정답은 항상 일반 영어 단어입니다 — 사람 이름, 지명, 브랜드명은 없습니다. 이로써 배경 지식에 관계없이 모든 플레이어에게 공정한 게임이 됩니다. {% else %} No. The answer will always be a common English word — no person names, place names, or brand names. {% if lang == 'ko' %} 데일리 모드에서 단어는 날짜 기반 알고리즘으로 선택됩니다 — 같은 단어가 오랫동안 반복되지 않습니다. 무제한 모드에서 단어는 무작위로 선택되므로 이론적으로 반복이 가능하지만, 단기적으로는 거의 일어나지 않습니다. {% else %} In Daily mode, words are selected using a date-based algorithm — the same word won't repeat for a very long time. {% if lang == 'ko' %} 통계는 브라우저의 localStorage에 저장됩니다. 다음 경우 잃을 수 있습니다:
