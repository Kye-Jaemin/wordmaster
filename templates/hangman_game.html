{% if lang == 'ko' %}학습 도구로서의 행맨{% else %}About hangman as a learning tool{% endif %}

{% if lang == 'ko' %}행맨은 백 년 넘게 종이와 연필로 즐겨온 게임입니다. 그 이유 중 하나는 — 게임 메카닉 자체가 영어 글자 빈도를 학습시키기 때문입니다. 플레이어는 자연스럽게 정보값이 큰 모음과 흔한 자음(E, T, A, O, I, N, S, R)부터 추측하는 법을 익히게 됩니다.{% else %}Hangman has been a paper-and-pencil staple for over a century because the game mechanic happens to teach English letter frequency. Players quickly learn to guess vowels and common consonants (E, T, A, O, I, N, S, R) first, because those letters carry the most information at low cost.{% endif %}

{% if lang == 'ko' %}어휘 학습 관점에서 행맨이 가진 과소평가된 강점: 단어를 못 맞췄을 때, 정답이 본인이 시도한 글자(맞춘 것 + 틀린 것)와 함께 공개됩니다. 어떤 글자가 의외였는지가 눈에 보이고 — 그 작은 놀라움이 스펠링을 기억하게 만듭니다. 너무 친절한 단어 게임은 이런 종류의 견고한 기억을 만들지 못합니다.{% else %}For vocabulary learning, hangman has an underrated property: when you fail to guess a word, the answer is revealed alongside the letters you got right and wrong. You see exactly which letters were unexpected — and that surprise is what makes the spelling stick. Word games with too much hand-holding don't produce this kind of durable memory; the small sting of being wrong is the feature, not a bug.{% endif %}

{% if lang == 'ko' %}생명 6개는 전통일 뿐, 절대적 수치는 아닙니다. 더 어려운 버전을 선호하는 사람도 있습니다. 숫자보다 중요한 건 매일 플레이하는 것 — 계속하기만 하면 글자 빈도 직관은 끊임없이 다듬어집니다.{% else %}Six lives is a tradition, not a constant. Some prefer harder versions with fewer. The number matters less than playing daily — your letter-frequency intuition keeps sharpening as long as you keep playing.{% endif %}
