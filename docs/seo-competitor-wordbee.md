# 경쟁사 심화 — Word Bee & "워드 툴" 진영 (SEO 관점)

> 출처: 웹 검색(2026-06). 정밀 트래픽·백링크 수치는 SEO 툴 미연결로 *정성 분석*. 결론 먼저: **정면으로 솔버(cheat) SERP를 이기려 하지 말고, 그들의 프로그래매틱 플레이북을 WordMaster의 "학습" 각도로 변형해서 가져온다.**

---

## 1. 경쟁 지형 (의도별 2진영)

| 진영 | 대표 | 검색 의도 | WordMaster 적합성 |
|---|---|---|---|
| **A. 워드 툴/솔버** | Word Bee, WordHippo, word.tips, thefreedictionary, dictionary.com | "anagram solver", "word unscrambler", "5 letter words with…", "wordle helper" (=치트/도구) | **낮음** — 정면승부 비추천 |
| **B. 데일리 게임/학습** | NYT(Wordle/Spelling Bee/Connections), Semantle, 각종 vocab 앱 | "daily word game", "vocabulary game", "games like wordle" | **높음** — WordMaster의 본진 |

핵심: 상위 노출을 먹는 대형 사이트(Word Bee류)는 대부분 **A진영(도구)**이다. 이들은 거대한 프로그래매틱 페이지로 검색을 점유하지만, **"학습/기억"이 아니라 "정답 찾기(치트)" 의도**를 노린다. WordMaster의 제품 정체성(학습이 남는 게임)과 다르다.

## 2. Word Bee 분해 (thewordbee.com)

**구성**
- **데일리 게임 8종+** (Word Scramble = 매일 5개 5글자 단어 등) — no sign-up, 무료.
- **솔버 도구**: Anagram Solver / Word Unscrambler — 글자 입력 시 길이별로 가능한 단어 전부 출력, `?` 와일드카드(최대 2개) 지원. → Scrabble/Words With Friends/크로스워드/워들 치트 수요 흡수.
- **3.3만+ 단어 레퍼런스 페이지** + 전략 가이드.

**왜 SEO가 강한가 (= 우리가 배울 점)**
1. **프로그래매틱 규모**: 단어·글자조합마다 페이지 → 롱테일 무한 커버. (← WordMaster의 `/word/<word>` 설계안과 동일 원리)
2. **도구 = 자연 백링크/재방문**: 솔버는 반복 사용·공유됨.
3. **데일리 게임 = 신선도·재방문 신호**.
4. **마찰 0** (무가입·무설치).

**약점 (= 우리가 이길 곳)**
- 학습/기억 추적 없음(치트 도구는 "외우게" 하지 않음).
- 이중언어/한국 시장 무시.
- 광고 의존·도구 위주 → 체류·브랜드 충성도 약함.

## 3. What to COPY vs What to AVOID

**COPY (변형해서 도입)**
- ✅ **프로그래매틱 단어 페이지** — 단, 솔버가 아니라 **단어 학습 페이지**(`/word/<word>` 설계안). 정의+예문+어원+**플레이/내 단어장** = 도구가 못 주는 "기억" 가치.
- ✅ **"단어 리스트" 리소스** — "SAT 빈출 100단어", "5글자 단어 모음(학습용)", "동물 단어 리스트" 등. 워들발 `5 letter words` 수요를 **치트가 아니라 학습 리스트 + 플레이 CTA**로 흡수.
- ✅ **데일리 신선도** — 이미 데일리/아카이브 보유. 단어 페이지·리스트와 내부링크로 묶어 신선도 전파.

**AVOID (하지 말 것)**
- ❌ **솔버/언스크램블러 도구 페이지** — "anagram solver" SERP는 도메인 권위 큰 도구 사이트가 장악, 정면승부 비효율 + 제품 정체성("학습")과 충돌(치트 조장).
- ❌ 치트성 카피("정답 찾기/cheat") — 학습 브랜드 톤 훼손.

## 4. 헤드투헤드

| 항목 | WordMaster | Word Bee(A진영) | NYT 게임(B진영) |
|---|---|---|---|
| 프로그래매틱 페이지 수 | 적음(아카이브 30일) → **확장 예정** | 3.3만+ | 적음(게임 위주) |
| 핵심 의도 | 학습·기억 | 치트·도구 | 데일리 엔터 |
| 학습 추적 | ✅ 내 단어장·기록 | ✗ | ✗ |
| 이중언어(ko/en) | ✅ | ✗ | ✗ |
| 앱(리텐션) | ✅ Play | 대개 웹 | ✅ |
| 데일리 신선도 | ✅ | ✅ | ✅ |
| 도메인 권위 | 낮음(신생) | 높음 | 매우 높음 |

**해석**: 규모·권위로는 밀린다. 그러나 **"학습이 남는다 + 이중언어 + 앱"** 교집합은 A·B 어느 쪽도 안 채우는 빈틈. 여기에 프로그래매틱(학습형)으로 규모를 더하면 *경쟁이 약한 롱테일 + 한국 시장*에서 점유 가능.

## 5. 구체 플레이 (우선순위)

1. **`/word/<word>` 학습 페이지 프로그래매틱** — Word Bee의 규모 전술을 학습 각도로 복제. (→ `seo-word-pages-spec.md`) **P1·장기**
2. **학습용 단어 리스트 페이지** — "수능 빈출 100", "5글자 학습 단어", 카테고리 리스트. 워들발 수요를 학습 프레임으로 흡수. **P2**
3. **한국 시장 선점** — A·B 모두 무시하는 ko 키워드(수능·토익·영단어 게임). (→ `seo-korean-keyword-briefs.md`) **P1**
4. **"games like wordle / free wordle alternative" 비교 랜딩** — B진영 의도, 난이도 중간. 학습 차별점으로 포지셔닝. **P2**
5. **데일리↔단어페이지↔리스트 내부링크 허브** — 신선도·권위 내부 전파. **P1(저노력)**

## 6. 측정
- GSC에서 진영별 쿼리 추적: 학습형("[word] meaning", "수능 영단어") vs 게임형("daily word game", "games like wordle").
- **솔버 쿼리("anagram solver")는 일부러 목표에서 제외** — 추적 지표에서도 제외해 자원 분산 방지.

---

**한 줄 요약**: Word Bee는 *도구*로 규모를 만들어 이긴다. WordMaster는 같은 *프로그래매틱 규모*를 **학습·이중언어**라는, 그들이 비운 칸에 적용해야 이긴다 — 솔버 SERP는 쫓지 말 것.
