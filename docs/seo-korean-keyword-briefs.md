# 한국어 키워드 클러스터 — 콘텐츠 브리프

> 전략: 운영자가 한국어 기반 + 사이트가 이중언어인데 한국 검색 유입은 미개척. 한국 영단어 학습 키워드는 **경쟁이 영어권보다 약하고 수요가 명확**(수능·토익·ESL). 필러 1 + 클러스터 4 구조로 토픽 권위를 쌓는다.

## 클러스터 구조

```
[필러] 영어 단어 게임으로 어휘 외우기 (완전 가이드)        → /blog/korean-vocab-game-guide (신규, ko 중심)
   ├─ 수능 영단어 게임으로 외우기                          → /blog/suneung-vocab-game (신규)  + /high 강화
   ├─ 토익 영단어 게임으로 외우기                          → /blog/toeic-vocab-game (신규)   + /college 강화
   ├─ 워들 한국어로 즐기기 / 영어 워들이란?                → /blog/wordle-korean (신규)      + 홈 ko 강화
   └─ 중학 기초 영단어 게임 (ESL 입문)                     → /blog/basic-english-vocab-game (신규) + /middle 강화
```

내부링크 규칙: 필러 ↔ 4개 클러스터 상호 링크, 각 클러스터 → 해당 게임 카테고리 페이지(플레이 CTA), 게임 카테고리 페이지 → 클러스터(가이드 더보기).

---

## 공통 작성 규칙
- **언어**: 본문 한국어 우선, 영어 단어 예시 병기. ko 가독성(줄바꿈·문단) 유지(기존 피드백 규칙).
- **구조**: H1 1개, H2 4~6개, 도입부 100단어 안에 핵심 키워드.
- **분량**: 1,200~1,800자(한국어 기준), 얇은 콘텐츠 회피.
- **CTA**: 본문 중간·하단에 해당 게임 모드 플레이 버튼.
- **스키마**: 블로그는 기존 Article 스키마 자동 적용. 가이드성이면 HowTo 추가 고려.
- **내부링크**: 최소 3개(필러/형제 클러스터/게임 페이지).

## 브리프 1 — [필러] 영어 단어 게임으로 어휘 외우기
- **타겟 키워드**: `영어 단어 게임`, `영단어 게임`, `단어 외우기 게임` (정보+상업)
- **URL**: `/blog/korean-vocab-game-guide?lang=ko`
- **Title(≤60자)**: 영어 단어 게임으로 어휘 외우기 — 게임으로 영단어 마스터 | WordMaster
- **Meta(≤80자)**: 단어카드보다 덜 지루하게. 게임으로 영단어를 외우는 법, 수능·토익·기초까지 레벨별 가이드.
- **H2 구성**:
  1. 왜 게임으로 외우면 더 오래 남는가 (분산 반복·맥락 학습 근거)
  2. 어떤 게임이 어휘에 효과적인가 (타일/애너그램/행맨 비교)
  3. 레벨별 추천 (중학·수능·토익·GRE) — 각 클러스터로 링크
  4. 하루 5분 루틴 만들기 (데일리 챌린지 + 스트릭)
  5. 외운 단어를 잊지 않는 법 (내 단어장·복습)
- **핵심 내부링크**: 4개 클러스터 + /daily + /my-words
- **노력/우선순위**: 반나절 / **P1**

## 브리프 2 — 수능 영단어 게임
- **타겟 키워드**: `수능 영단어 게임`, `수능 영단어 외우기`, `수능 영어 단어` (상업)
- **URL**: `/blog/suneung-vocab-game?lang=ko` + `/high`(고등학생 수준) ko 카피 강화
- **Title**: 수능 영단어, 게임으로 외우기 — 고등 필수 어휘 단어 게임 | WordMaster
- **Meta**: 수능에 자주 나오는 영단어를 게임으로. 매일 5분, 단어카드 없이 어휘력 키우는 법.
- **H2 구성**:
  1. 수능 영단어, 왜 안 외워질까 (단순 암기의 한계)
  2. 게임으로 외우는 3단계 루틴
  3. WordMaster 고등 수준 모드 활용법 (→ /high 플레이 CTA)
  4. 자주 나오는 수능 빈출 어휘 예시 10개(+뜻)
  5. 복습: 내 단어장으로 약점 단어 관리
- **내부링크**: 필러, /high, /my-words, top-100-sat-words(기존 블로그)
- **노력/우선순위**: 반나절 / **P1** (경쟁 약·수요 명확)

## 브리프 3 — 토익 영단어 게임
- **타겟 키워드**: `토익 영단어`, `토익 단어 외우기`, `토익 빈출 단어` (상업)
- **URL**: `/blog/toeic-vocab-game?lang=ko` + `/college` ko 강화
- **Title**: 토익 영단어 외우기 — 빈출 어휘를 게임으로 | WordMaster
- **Meta**: 토익 빈출 영단어를 게임으로 익히기. 출퇴근 5분, 비즈니스 영어 어휘력 키우기.
- **H2 구성**:
  1. 토익 단어, 맥락 없이 외우면 잊는다
  2. 비즈니스/일상 빈출 어휘 게임으로 익히기
  3. WordMaster 대학·고급 모드 활용 (→ /college CTA)
  4. 토익 빈출 단어 예시 10개(+뜻·예문)
  5. 하루 루틴 + 복습 전략
- **내부링크**: 필러, /college, /my-words, ielts-essential-words(기존)
- **노력/우선순위**: 반나절 / **P2**

## 브리프 4 — 워들 한국어 / 영어 워들
- **타겟 키워드**: `워들 한국어`, `영어 워들`, `워들 게임`, `wordle 한국어` (정보)
- **URL**: `/blog/wordle-korean?lang=ko` + 홈 ko 카피
- **Title**: 영어 워들(Wordle) 한국어로 즐기기 — 무료 단어 추측 게임 | WordMaster
- **Meta**: 워들 스타일 영어 단어 게임을 한국어 안내로. 규칙·전략·매일 챌린지까지 무료.
- **H2 구성**:
  1. 워들이란? 규칙 한국어 설명 (→ /how-to-play 링크)
  2. WordMaster가 워들과 다른 점 (학습 추적·이중언어·3가지 게임)
  3. 첫 단어 전략 (CRANE/AUDIO/STARE)
  4. 매일 무료로 즐기는 법 (→ /daily CTA)
- **내부링크**: 필러, /daily, /how-to-play, best-starting-words(기존)
- **노력/우선순위**: 반나절 / **P1** (브랜드 인지 키워드, 전환 용이)

## 브리프 5 — 중학 기초 영단어 게임 (ESL 입문)
- **타겟 키워드**: `중학 영단어 게임`, `기초 영단어 외우기`, `초등 영어 단어 게임` (상업)

- **Title**: 기초·중학 영단어 게임 — 쉬운 단어부터 게임으로 | WordMaster
- **Meta**: 초·중학 기초 영단어를 게임으로 재미있게. 4글자 쉬움 모드부터 차근차근.
- **H2 구성**:
  1. 기초 어휘, 흥미가 먼저다
  2. 쉬움(4글자)·중학 모드 활용 (→ /easy, /middle CTA)
